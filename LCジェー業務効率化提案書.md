## 1. 現状分析

### 1.1 御社の事業概要

LCジェー様は、日本と中国にまたがるオンラインチームでライブコマース事業を展開されています。主な業務は以下の4つに分類されます。

| 業務カテゴリ | 主な内容 | 担当チーム |
|-------------|---------|-----------|
| **ブランド営業** | ブランドへの営業活動、商談、契約締結 | 営業チーム |
| **ライブコマース運営** | 配信企画、インフルエンサー管理、売上管理 | 運営チーム |
| **求人・採用** | インフルエンサー・スタッフの採用活動 | 人事チーム |
| **事務・管理** | 経理、契約書管理、各種事務作業 | 事務チーム |

### 1.2 現在のシステム機能

現在のシステムには以下の機能が実装されています。

| 機能カテゴリ | 実装済み機能 |
|-------------|-------------|
| タスク管理 | スクショ連携タスク作成、自動リマインド、完了追跡 |
| 日報管理 | 日報作成・閲覧、AI分析、フォローアップ追跡 |
| ブランド管理 | ブランド登録、商品管理、対応履歴、直播履歴 |
| 名刺管理 | AI OCR認識、重複チェック、担当者記録 |
| スタッフ管理 | 担当者名簿、レポートスタッフ、国別管理 |
| 多言語対応 | 日本語・中国語切り替え |

## 2. 業務効率化のための機能提案

御社の業務フローを分析した結果、以下の機能を優先度順に提案いたします。

### 2.1 【最優先】営業パイプライン管理機能

**課題**: ブランド営業の進捗状況が可視化されておらず、フォローアップ漏れが発生しやすい

**提案機能**:

```
営業パイプライン（カンバンボード形式）
┌─────────────┬─────────────┬─────────────┬─────────────┬─────────────┐
│  初回接触   │   商談中    │  見積提出   │  契約交渉   │   成約済    │
├─────────────┼─────────────┼─────────────┼─────────────┼─────────────┤
│ ブランドA   │ ブランドC   │ ブランドE   │ ブランドG   │ ブランドI   │
│ ブランドB   │ ブランドD   │ ブランドF   │             │ ブランドJ   │
└─────────────┴─────────────┴─────────────┴─────────────┴─────────────┘
```

**実装内容**:
- ブランドをドラッグ&ドロップでステージ移動
- 各ステージの滞留日数を自動計算・警告表示
- 名刺情報との自動連携（担当者の連絡先をワンクリックで表示）
- 営業担当者別の成約率・進捗レポート

**期待効果**: 営業活動の可視化により、フォローアップ漏れを80%削減

---

### 2.2 【最優先】ライブ配信スケジュール管理機能

**課題**: 複数のライブ配信スケジュールを管理し、インフルエンサー・商品・ブランドを効率的にアサインする必要がある

**提案機能**:

| 機能 | 説明 |
|------|------|
| カレンダービュー | 月間・週間のライブ配信スケジュールを一覧表示 |
| 配信枠管理 | 配信時間帯ごとにインフルエンサーと商品をアサイン |
| 自動リマインド | 配信24時間前・1時間前に関係者へ通知 |
| 売上実績連携 | 配信後の売上を自動集計してブランド別レポートに反映 |

**実装イメージ**:
```
2026年1月 ライブ配信スケジュール
┌────┬────────────────────────────────────────────────┐
│ 日 │ 月 │ 火 │ 水 │ 木 │ 金 │ 土 │
├────┼────┼────┼────┼────┼────┼────┤
│    │    │    │ 1  │ 2  │ 3  │ 4  │
│    │    │    │    │19:00│    │    │
│    │    │    │    │ブランドA│   │    │
├────┼────┼────┼────┼────┼────┼────┤
│ 5  │ 6  │ 7  │ 8  │ 9  │ 10 │ 11 │
│    │20:00│    │    │19:00│20:00│    │
│    │ブランドB│   │    │ブランドC│ブランドD│   │
└────┴────┴────┴────┴────┴────┴────┘
```

**期待効果**: 配信準備の効率化、ダブルブッキング防止

---

### 2.3 【高優先】インフルエンサー管理機能

**課題**: ライブコマースの要であるインフルエンサーの情報が一元管理されていない

**提案機能**:

| 管理項目 | 内容 |
|---------|------|
| 基本情報 | 名前、連絡先、SNSアカウント、得意ジャンル |
| 実績データ | 過去の配信回数、総売上、平均視聴者数、成約率 |
| スケジュール | 稼働可能日時、予約済み配信 |
| 報酬管理 | 報酬体系（固定/歩合）、支払い履歴 |
| 評価・メモ | 配信品質、コミュニケーション、改善点 |

**連携機能**:
- ライブ配信スケジュールとの自動連携
- ブランドとの相性マッチング（過去実績ベース）
- 報酬計算の自動化

**期待効果**: インフルエンサーの最適配置、報酬計算の工数削減

### 2.4 【高優先】売上・成果ダッシュボード

**課題**: ブランド別・インフルエンサー別・期間別の売上分析が手作業

**提案機能**:

```
┌─────────────────────────────────────────────────────────────┐
│                    売上ダッシュボード                        │
├─────────────────┬─────────────────┬─────────────────────────┤

│  ¥12,500,000   │  +15.2%        │  83.3%                  │
├─────────────────┴─────────────────┴─────────────────────────┤
│                                                             │
│  ブランド別売上TOP5          インフルエンサー別売上TOP5     │
│  1. ブランドA ¥3,200,000    1. 田中花子 ¥2,800,000        │
│  2. ブランドB ¥2,100,000    2. 李明 ¥2,100,000            │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**グラフ機能**:
- 日別・週別・月別の売上推移グラフ
- ブランド別売上構成比（円グラフ）
- インフルエンサー別パフォーマンス比較

**期待効果**: 経営判断の迅速化、売上目標管理の効率化

### 2.5 【中優先】求人・採用管理機能

**課題**: インフルエンサーやスタッフの採用プロセスが管理されていない

**提案機能**:

| ステージ | 管理内容 |
|---------|---------|
| 応募受付 | 応募者情報、応募経路、希望条件 |
| 書類選考 | 履歴書・ポートフォリオ管理、評価コメント |
| 面接 | 面接日程調整、面接評価、合否判定 |
| 採用決定 | オファー内容、契約条件、入社日 |
| オンボーディング | 研修進捗、必要書類、アカウント発行 |

**自動化機能**:
- 応募者への自動返信メール
- 面接リマインド通知
- 採用決定時の契約書テンプレート生成

**期待効果**: 採用業務の工数50%削減、応募者対応の迅速化

---

### 2.6 【中優先】契約書・請求書管理機能

**課題**: ブランドとの契約書、インフルエンサーへの報酬支払いが手作業

**提案機能**:

| 機能 | 説明 |
|------|------|
| 契約書テンプレート | ブランド契約、インフルエンサー契約のテンプレート管理 |

| 請求書自動生成 | 売上実績から請求書を自動作成 |
| 支払い管理 | インフルエンサーへの報酬支払いスケジュール管理 |

**期待効果**: 契約更新漏れ防止、経理業務の効率化
